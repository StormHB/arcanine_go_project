Data-driven Pokémon GO raid guide

Dynamic raids, boss pages and counter recommendations

Arcanine Go Project is a responsive Pokémon GO raid guide built around dynamic raid data, generated counter recommendations and dedicated boss detail pages. The site combines clean frontend design with a Pokebattler-based counter pipeline.

What this project includes

The project has evolved from a static page layout into a small data-driven raid guide with reusable rendering logic, generated counter data and support tooling.

01

Dynamic raid rotations

Raid schedules, summary cards and raid cards are rendered from structured JavaScript data. Active, upcoming and ended states are calculated automatically.

02

Boss detail pages

Each raid boss can open a dedicated detail page with typing, weaknesses, status, best counters, budget counters and move metadata.

03

Counter pipeline

Counter data is prepared through a scrape → parse → transform workflow using default Pokebattler simulations and reviewed before publishing.

04

Contact support

Feedback, raid requests and counter corrections can be submitted through a Formspree-powered contact form with built-in validation.

Project goals

The goal of Arcanine Go Project is to build a maintainable, responsive and accessible raid guide that demonstrates modern frontend structure without using a framework. The project focuses on reusable data, clean rendering logic, practical tooling and a polished user experience.
