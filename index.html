Pokemon GO raid guide for April 2026

Arcanine Go Project is a clean multi-page website focused on the current Mega and Tier 5 raid rotation. The site presents raid bosses, their weaknesses, and both best and budget counter options in a simple card-based layout.

What this project includes

The website is designed to satisfy the main faculty requirements while keeping the theme focused and easy to explain.

01

Raid overview

A dedicated raid page shows the active Mega and Tier 5 bosses for the current monthly rotation.

02

Counter cards

Every featured boss has a card with weaknesses, best counters and budget alternatives.

03

Responsive design

The layout adjusts to smaller screens and uses a CSS-only hamburger menu without JavaScript.

04

Accessible basics

The project includes semantic sections, a skip link, visible focus styles and alt text for images.
